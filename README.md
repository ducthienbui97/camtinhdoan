## Download
```
git clone https://github.com/ducthienbui97/camtinhdoan.git
cd camtinhdoan 
npm install
```

## Build app (for frontend)
``` npm run build ```

## Run app (for backend)
``` npm run start ```

## Build and run
``` npm run start:prod ```
